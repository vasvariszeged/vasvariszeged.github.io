# Az alapok

A __C#__ programozás világa előttünk hever, és csak arra vár, hogy felfedezzük. Ebben a részben megkezdjük kalandunkat, és megismerkedünk a __C#__ nyelven történő programozás alapjaival:

- Megtanuljuk, mi a __C#__ és a __.NET__
- Telepítjük a szükséges eszközöket a __C#__ programozáshoz 
- Megírjuk az első programunkat, és megismerjük a __C#__ program alapvető összetevőit 
- Megjegyzések létrehozása a kódban
- Adatok tárolása változókban
- A típusrendszer megértése
- Alapvető matematikai feladatok
- Bemeneti adatokat használata
- Döntések meghozatala
- A kód futtatása ciklusokban
- Több adatot tartalmazó tömbök készítése
- Metódusok, az-az kódrészletek újrahasználhatóságának alkalmazása
- A memória használatának megértése a __C#__ nyelvben
