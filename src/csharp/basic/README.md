# Az alapok 1. rész

A __C#__ programozás világa előttünk hever, és csak arra vár, hogy felfedezzük. Ebben a részben megkezdjük tanulmányainkat és megismerkedünk a __C#__ nyelven történő programozás alapjaival:

- Megtanuljuk mi a __C#__ és a __.NET__
- Telepítjük a szükséges eszközöket a __C#__ programozáshoz 
- Megírjuk az első programunkat, és megismerjük a __C#__ program alapvető összetevőit 
- Kommentek létrehozása a kódban
- Adatok tárolása változókban
- A típusrendszer megértése
- Alapvető matematikai feladatok
